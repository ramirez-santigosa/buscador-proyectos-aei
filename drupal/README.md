# Módulo Drupal: buscador_aei

Plantilla de módulo Drupal 9/10 para integrar el Buscador de Proyectos AEI
como bloque en la web institucional de la AEI.

**Estado:** pendiente (Bloque 7). El frontend HTML/JS debe estar terminado primero.

## Integración

1. Copiar la carpeta `drupal/` al directorio de módulos custom de Drupal.
2. Cambiar `API_BASE_URL` en `app.js` a la URL del backend desplegado.
3. Activar el módulo y colocar el bloque `buscador_aei` en la región deseada.
